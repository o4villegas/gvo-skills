## How to Use These Skills

Each subfolder represents a corporate role. Inside each role folder, you'll find individual skill folders containing a `SKILL.md` file. To use a skill:

1. **In Claude Code:** Copy the skill folder to `~/.claude/skills/` or your project's `.claude/skills/` directory
2. **In Cowork:** Skills in the selected folder are automatically available
3. **Manual use:** Open any SKILL.md and follow the instructions directly
